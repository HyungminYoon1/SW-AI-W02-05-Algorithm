4. 인덱스를 이용해서 배열의 특정 값을 읽을 때 시간 복잡도를 빅오 표기법으로 기입하라.  또한 정답의 이유를 설명하라.

# 답변

O(1) (상수 시간)


## 이유
배열(Array)은 연속된 메모리 공간에 저장되는 자료구조입니다.
따라서 인덱스를 이용한 접근은 다음과 같은 방식으로 계산됩니다.

배열의 시작 주소를 base_address 라고 하면
address(i)=base_address+(i×element_size)

즉,
1. 배열의 시작 주소를 알고 있음
2. 원소의 크기(element size)가 일정함
3. 인덱스 i 가 주어지면 곱셈 1번 + 덧셈 1번으로 바로 주소 계산 가능


예를 들어:
arr = [10, 20, 30, 40, 50]
x = arr[3]   # 40

CPU는 다음과 같이 접근합니다. 
base_address + (3 * element_size)

따라서 배열 길이 n과 관계없이 항상 동일한 연산 횟수 즉 O(1)
