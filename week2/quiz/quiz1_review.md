# 답변

## 1. 문자열(str)을 그대로 사용하는 방법
Python의 문자열은 immutable(불변 객체)입니다.
즉, 문자열이 생성된 이후 내용을 직접 수정할 수 없습니다.

s + "!" 은 기존 문자열을 수정하는 것이 아니라

1. 새로운 메모리 공간을 생성
2. 기존 문자열 복사
3. "!" 추가
4. 새로운 문자열 반환

이라는 과정을 수행합니다.

장점
1) 코드가 간단하고 직관적
2) 메모리 안정성
3) 문자열 처리 함수가 풍부

단점
1) 문자열 수정 비용이 큼
2) 부분 수정이 불가능

## 2. 문자열을 리스트로 변환하여 사용하는 방법

Python 리스트는 mutable(가변 객체)입니다.
즉, 내용을 직접 수정할 수 있습니다.

장점
1) 리스트는 요소 변경이 가능합니다. *lst[i] = x 의 시간 복잡도: O(1)
2) 반복적인 추가 작업이 효율적
3) 문자열 조작에 유리 (문자 삽입 / 삭제 / 교체 등)

단점
1) 리스트 → 문자열 변환시 마지막에 join() 필요
2) 코드가 길어짐
3) 메모리 사용 증가
* 리스트는 각 요소가 객체 reference 이므로, 문자열보다 메모리 사용량이 큽니다.
