# 2차원 리스트 (배열 회전)

## 개요

N×N 크기의 **2차원 리스트**(2중 리스트)를 시계방향으로 90도 회전시키는 인덱스 변환 규칙을 학습합니다. 파이썬에서는 `[[1,2],[3,4]]` 형태의 리스트로 2차원 구조를 표현합니다.

## 회전 규칙

시계방향 90도 회전 시 위치 변환:

```
(i, j) → (j, n-1-i)
```

- 원본 `matrix[i][j]` → 회전 후 `rotated[j][n-1-i]`
- **유도**: 시계 90도 회전 시 원래 i행이 결과의 (n-1-i)열이 되고, 원래 j열이 결과의 j행이 됨

## 예시

```
원본 (3×3):          회전 후:
[1, 2, 3]            [7, 4, 1]
[4, 5, 6]     →      [8, 5, 2]
[7, 8, 9]            [9, 6, 3]
```

## 구현 흐름

1. n×n 크기의 새 배열 생성
2. `(i, j)`에 있는 원소를 `rotated[j][n-1-i]`에 복사

## 다른 각도 회전

| 회전 | 변환 규칙 |
|------|-----------|
| 시계 90° | (i, j) → (j, n-1-i) |
| 시계 180° | (i, j) → (n-1-i, n-1-j) |
| 시계 270° (반시계 90°) | (i, j) → (n-1-j, i) |

## 활용 예시

- 이미지 회전
- 매트릭스 조작 문제
- 패턴 회전/대칭
