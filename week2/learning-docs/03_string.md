# 문자열 (회문 판별)

## 개요

**회문(Palindrome)**은 앞에서 읽으나 뒤에서 읽으나 같은 문자열입니다.  
대소문자 구분 없이, 공백과 특수문자를 제외한 부분만 비교합니다.

## 판별 과정

1. **전처리**: 알파벳·숫자만 남기고 소문자로 통일
   - `isalnum()`: 알파벳 또는 숫자인지 확인
   - `lower()`: 소문자로 변환

2. **비교**: 정제된 문자열이 회문인지 확인
   - 방법 1: `refined_s == refined_s[::-1]` (뒤집어서 비교)
   - 방법 2: 양 끝에서 시작하는 투 포인터

## 투 포인터 방식

```
left = 0, right = len(s)-1
left < right 동안:
  s[left] == s[right]면 left++, right--
  다르면 False
모두 일치하면 True
```

## 시간 복잡도

- O(n): 문자열 순회 및 비교
