# 버블 정렬 (Bubble Sort)

## 개요

**버블 정렬**은 인접한 두 원소를 비교해 큰 값을 오른쪽으로 밀어내며 정렬하는 방식입니다.  
가장 큰 원소가 거품처럼 맨 끝으로 올라갑니다.

## 동작 방식

1. 인접한 두 원소 `arr[j]`, `arr[j+1]` 비교
2. `arr[j] > arr[j+1]`이면 교환
3. 한 패스마다 가장 큰 원소가 끝으로 이동
4. 패스를 n-1번 반복

## 의사코드

```
for i = 0 to n-2:
  for j = 0 to n-2-i:    // 이미 정렬된 끝부분 제외
    if arr[j] > arr[j+1]:
      swap(arr[j], arr[j+1])
```

## 최적화 (조기 종료)

- 한 패스에서 교환이 한 번도 없으면 이미 정렬된 상태
- `swapped` 플래그를 두고, `False`이면 바로 종료

## 시간 복잡도

| 구분 | 복잡도 |
|------|--------|
| 평균/최악 | O(n²) |
| 최선 (이미 정렬) | O(n) - 최적화 시 |

## 공간 복잡도

- O(1) (제자리 정렬)
