# 삽입 정렬 (Insertion Sort)

## 개요

**삽입 정렬**은 앞쪽의 정렬된 부분에 새 원소를 적절한 위치에 삽입해 정렬하는 방식입니다.  
카드 한 장씩 손에 들고 올바른 자리에 끼워 넣는 것과 비슷합니다.

## 동작 방식

1. 첫 번째 원소는 이미 정렬된 구간으로 간주
2. 두 번째 원소부터 차례로 선택 (key)
3. key를 앞의 정렬된 구간에서 맞는 자리에 삽입
4. key보다 큰 원소들은 한 칸씩 오른쪽으로 이동

## 의사코드

```
for i = 1 to n-1:
  key = arr[i]
  j = i - 1
  while j >= 0 and arr[j] > key:
    arr[j+1] = arr[j]
    j--
  arr[j+1] = key
```

## 시간 복잡도

| 구분 | 복잡도 |
|------|--------|
| 평균/최악 | O(n²) |
| 최선 (이미 정렬) | O(n) |

이미 정렬된 배열에서는 비교만 하고 이동이 거의 없어 O(n)에 가깝게 동작합니다.

## 공간 복잡도

- O(1) (제자리 정렬)

## 버블 정렬과 비교

- 둘 다 O(n²)이지만, 삽입 정렬은 이미 어느 정도 정렬된 데이터에서 더 유리함
