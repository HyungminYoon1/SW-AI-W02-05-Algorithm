# 우선순위 큐 (Priority Queue)

## 개요

**우선순위 큐**는 각 요소에 우선순위가 부여되어, 우선순위가 높은 요소가 먼저 꺼내지는 자료구조입니다. 일반 큐와 달리 FIFO가 아니라 우선순위에 따라 순서가 결정됩니다.

## 핵심 원리

- **Insert**: 우선순위와 함께 요소 추가
- **Delete**: 우선순위가 가장 높은 요소 제거
- 우선순위는 숫자로 표현 (작을수록 높은 경우: min-heap, 클수록 높은 경우: max-heap)

## 구현 방식

파이썬에서는 **heapq** 모듈을 사용합니다. heapq는 최소 힙(min-heap)으로, 가장 작은 값이 먼저 나옵니다.

```python
import heapq

heap = []
heapq.heappush(heap, (우선순위, 데이터))  # 삽입
우선순위, 데이터 = heapq.heappop(heap)     # 삭제 (최소값)
```

## 응급실 예시

```
숫자가 작을수록 우선순위 높음 (1 > 2 > 3)
[("김철수", 3), ("이영희", 1), ("박민수", 2)]
→ 처리 순서: 이영희 → 박민수 → 김철수
```

## 시간 복잡도

| 연산 | 복잡도 |
|------|--------|
| Insert | O(log n) |
| Delete (최우선) | O(log n) |
| Peek (최우선 조회) | O(1) |

## 활용 예시

- 응급실 환자 처리
- 작업 스케줄링 (우선순위 작업)
- 다익스트라 알고리즘
- 허프만 코딩

## 일반 큐와 비교

| 구분 | 큐 | 우선순위 큐 |
|------|-----|-------------|
| 순서 | FIFO | 우선순위 |
| 구현 | deque | heap |
