# 스택 (Stack)

## 개요

**스택**은 LIFO(Last In First Out, 후입선출) 구조의 자료구조입니다. 가장 나중에 넣은 데이터가 가장 먼저 꺼내집니다.

## 핵심 원리

- **Push**: 스택의 맨 위에 데이터 추가
- **Pop**: 스택의 맨 위에서 데이터 제거
- **Top/Peek**: 맨 위 데이터 조회 (제거하지 않음)

## 비유

접시를 쌓는 것과 같습니다. 위에 새 접시를 올리고(push), 맨 위 접시부터 꺼냅니다(pop).

## 파이썬 구현

```python
stack = []        # 리스트로 스택 구현
stack.append(x)   # push
stack.pop()       # pop (마지막 요소 제거)
stack[-1]         # top (peek)
```

## 시간 복잡도

| 연산 | 복잡도 |
|------|--------|
| Push | O(1) |
| Pop | O(1) |
| Peek | O(1) |

## 활용 예시

- 괄호 짝 맞추기
- 역순 처리 (문자열 뒤집기)
- 후위 표기법 계산
- DFS(깊이 우선 탐색)
- undo/redo 기능

## 괄호 짝 맞추기 흐름

```
'(' 만나면 → push
')' 만나면 → 스택이 비어있으면 False, 아니면 pop
반복 끝난 후 → 스택이 비어있으면 True
```
