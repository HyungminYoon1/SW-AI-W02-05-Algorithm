# 머지 정렬 (Merge Sort)

## 개요

**머지 정렬**은 분할 정복 방식을 사용하는 정렬 알고리즘입니다. 배열을 절반으로 나누고, 각각을 정렬한 뒤 두 정렬된 부분을 병합(merge)합니다.

## 핵심 원리

1. **분할 (Divide)**: 배열을 정확히 절반으로 나눔
2. **정복 (Conquer)**: 각 절반을 재귀적으로 정렬 (원소 1개가 될 때까지)
3. **병합 (Merge)**: 두 개의 정렬된 부분 배열을 합쳐 하나의 정렬된 배열로 만듦

## Merge 과정

```
왼쪽 배열 left_arr, 오른쪽 배열 right_arr
두 배열의 앞부터 비교하면서 작은 값을 결과 배열에 추가
한쪽이 비면 나머지 원소들을 순서대로 추가
```

## 시간 복잡도

| 구분 | 복잡도 |
|------|--------|
| 평균/최선/최악 | O(n log n) |

어떤 경우에도 O(n log n)으로 안정적입니다.

## 공간 복잡도

- O(n) - 병합을 위한 임시 배열 사용

## 장단점

**장점**
- 최악의 경우에도 O(n log n) 보장
- 안정 정렬 (같은 값의 상대적 순서 유지)

**단점**
- 추가 메모리 필요 (O(n))

## 퀵 정렬과 비교

| 구분 | 퀵 정렬 | 머지 정렬 |
|------|---------|-----------|
| 최악 | O(n²) | O(n log n) |
| 공간 | O(log n) | O(n) |
| 안정성 | 불안정 | 안정 |
