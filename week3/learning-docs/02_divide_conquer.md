# 분할 정복 (Divide and Conquer)

## 개요

**분할 정복**은 문제를 더 작은 하위 문제로 나누어 해결하고, 그 결과를 결합하여 원래 문제의 해를 구하는 알고리즘 설계 기법입니다.

## 핵심 원리

1. **분할 (Divide)**: 문제를 더 풀기 쉬운 작은 하위 문제로 나눔
2. **정복 (Conquer)**: 각 하위 문제를 재귀적으로 해결 (base case에 도달할 때까지)
3. **결합 (Combine)**: 하위 문제의 해를 합쳐 원래 문제의 해를 구함

## Base Case

재귀의 종료 조건. 보통 원소가 1개일 때 해당 값을 반환합니다.

```
예: 배열 최댓값 찾기
- left == right → arr[left] 반환
```

## 대표적 분할 정복 알고리즘

| 알고리즘 | 분할 방식 | 결합 방식 |
|----------|-----------|-----------|
| 이분 탐색 | 절반으로 나눔 | 한쪽만 선택 |
| 퀵 정렬 | 피벗 기준 분할 | 이미 분할 시 정렬됨 |
| 머지 정렬 | 정확히 절반 분할 | 두 정렬된 배열 병합 |

## 시간 복잡도

문제마다 다르며, 하위 문제의 개수와 결합 비용에 따라 결정됩니다.

- 최댓값 찾기: O(n)
- 머지 정렬: O(n log n)
- 퀵 정렬: 평균 O(n log n), 최악 O(n²)

## 활용 예시

- 배열의 최댓값/최솟값 찾기
- 퀵 정렬, 머지 정렬
- 이진 탐색
- 큰 수 곱셈 (카라추바 알고리즘)
