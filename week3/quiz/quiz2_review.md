# 작성한 응답
파이썬에는 기본적으로 최소 힙만 구현되어 있습니다. 
힙에 집어넣을 모든 원소에 - 를 붙여서 파싱합니다. 이 때 원소의 정렬  순서는 원래 순서의 반대가 됩니다. 꺼낼 때는 최솟값인 0번째 원소를 꺼내고 다시 -를 붙여서 원래의 값으로 복구합니다.
