# Week 5 학습 문서

동적 계획법(DP)과 그리디 알고리즘은 처음에는 비슷해 보이지만, 문제를 푸는 사고 방식이 꽤 다릅니다. 이 문서 모음은 두 주제를 구분해서 이해하고, 어떤 문제에서 어떤 접근을 선택해야 하는지 설명할 수 있도록 돕기 위해 정리했습니다.

## 목차

| 문서 | 설명 |
|------|------|
| [01. 동적 계획법](./01_dynamic_programming.md) | 큰 문제를 작은 문제로 나누고, 중복 계산을 줄여 최적해를 구하는 방법 |
| [02. 그리디 알고리즘](./02_greedy.md) | 매 단계에서 가장 좋아 보이는 선택을 하며 답을 만드는 방법 |

## 추천 학습 순서

1. 동적 계획법: 상태, 점화식, 메모이제이션과 테이블 채우기 방식을 먼저 익힙니다.
2. 그리디 알고리즘: 지역 최적 선택이 전체 최적해로 이어지는 조건을 이해합니다.
3. 두 방식 비교: 같은 최적화 문제라도 왜 어떤 문제는 DP로, 어떤 문제는 그리디로 푸는지 정리합니다.
