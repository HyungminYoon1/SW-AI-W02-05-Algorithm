# DP / Greedy 예상 기출문제

시험 대비용으로 **DP / Greedy 예상 기출문제 세트**를 정리했습니다.  
실전처럼 풀 수 있도록 **개념형 + 서술형 + 구현형**을 섞었습니다.

---

## 1. 개념 구분 문제
다음 문제들이 **DP**로 푸는 것이 적절한지, **Greedy**로 푸는 것이 적절한지 판단하고 이유를 쓰시오.

1. 동전을 사용해 합 `K`를 만드는 최소 동전 개수 구하기  
2. 회의실 사용 시간이 주어질 때 최대한 많은 회의 배정하기  
3. 계단을 오를 때 한 번에 1칸 또는 2칸씩 올라가며, 각 칸의 점수 합의 최댓값 구하기  
4. 여러 작업의 마감 시간이 주어질 때 지각을 최소화하도록 작업 순서 정하기  
5. 길이가 `N`인 수열에서 증가하는 부분수열의 최대 길이 구하기

1. 거스름돈 문제에서 화폐 단위가 `{500, 100, 50, 10}`일 때  
2. 화폐 단위가 `{1, 3, 4}`일 때  
3. 회의실 배정 문제에서 “끝나는 시간이 가장 빠른 회의”를 먼저 선택하는 전략

## 3. DP 점화식 세우기
정수 배열 `A[1...N]`이 주어진다.  
연속하지 않은 원소들만 선택할 수 있을 때, 선택한 원소 합의 최댓값을 구하시오.

1. 상태 정의를 하시오.  
2. 점화식을 세우시오.  
3. 초기값을 정하시오.  
4. 시간복잡도를 쓰시오.  

---

## 4. 0/1 Knapsack 유형
물건이 `N`개 있고, 각 물건은 무게 `w[i]`, 가치 `v[i]`를 가진다.  
가방의 최대 허용 무게가 `K`일 때 얻을 수 있는 최대 가치를 구하시오.

1. 이 문제가 Greedy로 잘 풀리지 않는 이유를 설명하시오.  
2. DP 상태를 정의하시오. 
3. 점화식을 작성하시오.  
4. 2차원 DP를 1차원 DP로 줄일 때 순회를 뒤에서 해야 하는 이유를 설명하시오.

3. 왜 이 문제에서는 Greedy가 항상 맞지 않는가?  
4. DP로 푼다면 점화식은?

## 6. 활동 선택 문제

3. 종료 시간이 같은 경우 어떤 기준을 추가해도 되는가?

# 실전 미니 모의고사

## A. OX 문제
1. DP 문제는 항상 재귀로만 풀어야 한다.  
2. Greedy는 항상 최적해를 보장하지 않는다.  
3. 최적 부분 구조가 있으면 항상 Greedy로 풀 수 있다.  
4. 0/1 Knapsack은 일반적으로 DP로 푼다.  
5. 회의실 배정 문제는 대표적인 Greedy 문제다.  
6. LIS는 대표적인 DP 문제다.
