# 문제

5. 트리 구조가 컴퓨터 시스템에서 활용되는 예를 두 가지 제시하고, 각각 해당 구조가 적합하다고 생각하는 이유를 서술하시오.

---

# 올바른 답안

트리 구조가 활용되는 예로는 파일 시스템의 디렉터리 구조와 데이터베이스의 인덱스 구조를 들 수 있습니다.

- 파일 시스템은 폴더 안에 하위 폴더와 파일이 포함되는 계층적 구조이므로 트리로 표현하기에 적합합니다.
- 데이터베이스 인덱스는 B-트리와 같은 구조를 사용하여 데이터를 정렬된 상태로 관리하고, 탐색·삽입·삭제를 효율적으로 수행할 수 있기 때문에 트리 구조가 적합합니다.
