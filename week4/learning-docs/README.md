# Week 4 학습 문서

트리와 그래프를 처음 배울 때 필요한 핵심 개념을 쉽게 이해하고, 다른 사람에게 설명할 수 있도록 정리한 문서 모음입니다.

## 목차

| 문서 | 설명 |
|------|------|
| [01. 이진 트리](./01_binary_tree.md) | 트리의 기본 구조와 전위, 중위, 후위 순회 |
| [02. 이진 검색 트리](./02_bst.md) | 정렬 성질을 가진 트리와 빠른 검색 원리 |
| [03. 그래프 기본](./03_graph_basic.md) | 그래프의 구성 요소와 인접 리스트 표현 |
| [04. BFS](./04_bfs.md) | 가까운 정점부터 탐색하는 너비 우선 탐색 |
| [05. DFS](./05_dfs.md) | 한 방향으로 깊게 들어가는 깊이 우선 탐색 |
| [06. 위상 정렬](./06_topological_sort.md) | 선행 관계를 만족하는 방향 그래프 정렬 |

## 추천 학습 순서

1. 이진 트리: 트리의 기본 구조와 순회를 익힙니다.
2. 이진 검색 트리: 트리 안에서 검색이 빨라지는 이유를 이해합니다.
3. 그래프 기본: 정점과 간선, 방향성, 인접 리스트를 정리합니다.
4. BFS와 DFS: 그래프 탐색의 두 가지 대표 방식을 비교하며 배웁니다.
5. 위상 정렬: 방향 그래프의 선후 관계를 처리하는 방법을 익힙니다.
